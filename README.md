# brahin-slate-editor

An editor for [Brahin](https://brahin.online/),
using the [Slate](https://www.slatejs.org/) library.

## Deployment

The `release` script builds the module, copies it into `../homepage/public_html/modules`
and executes `../homepage/bin/release`.
